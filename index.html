Compile time verification and generation of documentation from Xcode Playgrounds

Learn more

Verification

Eases the creation of Xcode Playgrounds with support for 3rd-party libraries.

Compile

Compiles Xcode Playgrounds with support for third party libraries from the command line.

Generation

Generates Playground Books from Swift packages and Markdown files from Xcode Playground.

Integration

Integrates Xcode Playgrounds with Carbon to export code snippets and Jekyll to create a microsite.
